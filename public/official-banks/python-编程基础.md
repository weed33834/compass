## 单选题

Python 中以下哪个是不可变数据类型？

A. list
B. dict
C. tuple
D. set

A. [1, 2, 3]
B. [2, 3]

## 单选题

Python 中定义类时，哪个方法会在创建实例时自动调用以初始化对象属性？

A. __init__
B. __new__
C. __call__
D. __create__

## 单选题

关于 Python 中的 is 和 == 运算符，下列说法正确的是？

A. is 比较值是否相等，== 比较身份是否相同
B. is 比较身份是否相同，== 比较值是否相等
C. is 和 == 完全等价
D. == 只能用于基本类型

答案：B
解析：is 比较两个对象的身份（id()，即内存地址）是否相同，== 比较两个对象的值是否相等（通过 __eq__ 方法）。例如 [1,2] == [1,2] 为 True，但 [1,2] is [1,2] 为 False。
难度：3
知识点：基础语法, 运算符, is vs ==
来源：Python 官方文档

## 单选题

关于 Python 的 GIL（全局解释器锁），下列说法正确的是？

A. GIL 使 Python 无法执行多线程
B. GIL 保证同一时刻只有一个线程执行 Python 字节码
C. GIL 在所有 Python 实现中都存在
D. GIL 无法被释放

答案：B
解析：GIL 是 CPython 的实现细节，保证同一时刻只有一个线程执行 Python 字节码。I/O 操作会释放 GIL，因此多线程适合 I/O 密集型任务。Jython、PyPy 等实现不一定有 GIL。
难度：4
知识点：Python-并发, GIL, 最佳实践
来源：Python 官方文档

使用 open() 打开文件时，下列哪些模式字符串表示可读可写？

## 判断题

Python 函数参数传递是“按对象的引用传递”，在函数内修改可变对象（如列表）会影响外部传入的原始对象。

答案：正确
解析：Python 中一切皆对象，函数参数传递的是对象的引用。修改可变对象（如 append 列表）会影响原对象；重新赋值则不会，因为只是让形参指向了新对象。
难度：2.5
知识点：函数, 参数传递
来源：Python 官方文档

## 判断题

闭包是指能够读取其他函数内部变量的函数，被读取的变量在其外部函数返回后仍然存活。

答案：正确

难度：3
知识点：函数, 闭包
来源：Python 官方文档

## 判断题

生成器函数使用 yield 关键字返回值，每次调用 next() 时执行到下一个 yield 处暂停。

答案：正确
解析：生成器是惰性求值的迭代器，yield 产出值并暂停函数状态，next() 恢复执行。生成器节省内存，适合处理大数据流。
难度：3
知识点：控制流, 生成器, 函数
来源：Python 官方文档
